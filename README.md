# Моделирование концентрационных полей при утечке в замкнутом помещении

Этот проект моделирует распространение вещества от локальной утечки в 2D-прямоугольной области и показания течеискателя, движущегося по траектории сканирования.

## 1. Основное уравнение модели

Используется уравнение адвекции-диффузии с источником:

```math
\frac{\partial C}{\partial t} = D\left(\frac{\partial^2 C}{\partial x^2} + \frac{\partial^2 C}{\partial y^2}\right) - u_x\frac{\partial C}{\partial x} - u_y\frac{\partial C}{\partial y} + Q\,S(x,y) - k_{\text{vent}}C
```

где:

- $C(x,y,t)$ — концентрация вещества,
- $D$ — коэффициент диффузии,
- $u_x, u_y$ — компоненты скорости потока воздуха,
- $Q$ — мощность источника утечки,
- $S(x,y)$ — пространственное распределение источника;
- $k_{\text{vent}}$ — коэффициент удаления вещества вентиляцией.

## 2. Источник утечки

Источник задан гауссовым распределением:

```math
S(x,y) = \exp\left(-\frac{(x-x_{\text{leak}})^2 + (y-y_{\text{leak}})^2}{2\sigma^2}\right)
```

Далее выполняется нормировка:

```math
S \leftarrow \frac{S}{\sum\limits_{i,j} S_{i,j}}
```

Это делает интегральную подачу вещества менее чувствительной к размеру расчетной сетки.

## 3. Пространственная дискретизация

Для внутренних узлов применяются центральные разности:

```math
\frac{\partial C}{\partial x}\Big|_{i,j} \approx \frac{C_{i,j+1}-C_{i,j-1}}{2\,\Delta x},
\qquad
\frac{\partial C}{\partial y}\Big|_{i,j} \approx \frac{C_{i+1,j}-C_{i-1,j}}{2\,\Delta y}
```

```math
\frac{\partial^2 C}{\partial x^2}\Big|_{i,j} \approx \frac{C_{i,j+1}-2C_{i,j}+C_{i,j-1}}{\Delta x^2},
\qquad
\frac{\partial^2 C}{\partial y^2}\Big|_{i,j} \approx \frac{C_{i+1,j}-2C_{i,j}+C_{i-1,j}}{\Delta y^2}
```

## 4. Шаг по времени (явная схема Эйлера)

Обновление концентрации на шаге $n \to n+1$:

```math
C^{n+1} = C^n + \Delta t\left[D\left(\frac{\partial^2 C}{\partial x^2}+\frac{\partial^2 C}{\partial y^2}\right)-u_x\frac{\partial C}{\partial x}-u_y\frac{\partial C}{\partial y}+Q\,S-k_{\text{vent}}C\right]
```

## 5. Ограничение шага по времени (устойчивость)

Используются оценки:

```math
\Delta t_{\text{diff}} = 0.25\,\frac{\min(\Delta x^2,\Delta y^2)}{D_{\text{max}}}
```

```math
\Delta t_{\text{adv}} = 0.45\,\frac{1}{\max\left(\frac{|u_x|}{\Delta x},\frac{|u_y|}{\Delta y},\varepsilon\right)}
```

```math
\Delta t = \min(\Delta t_{\text{diff}},\Delta t_{\text{adv}})
```

(в коде $\varepsilon=10^{-12}$ для защиты от деления на ноль, а $D_{\text{max}}$ берется с учетом наибольшего множителя диффузии в текущем сценарии).

## 6. Граничные условия

Для непроницаемых стен принимается условие Неймана:

```math
\frac{\partial C}{\partial n}=0
```

В численной реализации это задается копированием ближайших внутренних значений на границы.

## 7. Модель показаний датчика

Концентрация в текущей точке прибора $(x_s,y_s)$ вычисляется билинейной интерполяцией по 4 соседним узлам:

```math
C_s = (1-t_x)(1-t_y)C_{00} + t_x(1-t_y)C_{10} + (1-t_x)t_yC_{01} + t_xt_yC_{11}
```

где $t_x, t_y \in [0,1]$ — локальные координаты внутри ячейки сетки.

## 8. Сценарный анализ и отдельные графики различий

Скрипт `leak_detector_simulation.m` теперь запускает не один расчет, а набор сценариев. Для каждой группы ситуаций создается отдельное окно с графиками, на которых видна разница между режимами:

1. **Расход утечки** — сравниваются малый, базовый и повышенный расходы `Q`. На графиках отображаются площадь опасной зоны, максимальная концентрация, время первого появления опасной зоны и разность итоговых полей концентрации.
2. **Производительность вентиляции** — сравниваются слабая, средняя и сильная вентиляция через коэффициент `k_vent`. Для демонстрации выхода на безопасный режим утечка прекращается после 120 с, после чего оценивается скорость очистки помещения.
3. **Воздушные потоки** — сравниваются отсутствие направленного потока, равномерный поток, диагональный поток и локальный сквозняк. Карты концентрации строятся вместе с векторами скорости воздуха, а нижний ряд показывает разницу относительно режима без направленного потока.
4. **Коэффициент диффузии** — сравниваются пониженный, базовый и повышенный множители `D`. Графики показывают, что увеличение коэффициента диффузии приводит к более быстрому распространению вещества по помещению и выравниванию концентрационного поля.
5. **Случайные возмущения** — отдельно анализируются открывание двери, изменение направления воздушного потока, локальный сквозняк, изменение температуры и изменение внешнего давления. Для каждого возмущения строятся график средней концентрации, график разницы максимальной концентрации и карта разницы итогового поля.

Такой формат позволяет не накладывать все эффекты на один рисунок, а рассматривать каждую физическую ситуацию отдельно.

## 9. Модель вентиляции

Вентиляция учитывается линейным стоком:

```math
-k_{\text{vent}}C
```

При увеличении `k_vent` вещество быстрее удаляется из помещения. Поэтому средняя концентрация становится ниже, а после прекращения утечки система быстрее выходит на безопасный режим.

## 10. Влияние коэффициента диффузии

Для отдельной ситуации задаются сценарии с множителем `diffusionMultiplier`: `0.55D_0`, `D_0` и `2.20D_0`. В окне графика сравниваются:

- площадь области выше опасного порога — она растет быстрее при большем `D`;
- показатель неоднородности `max(C) - mean(C)` — он быстрее уменьшается при повышенной диффузии;
- итоговые контуры опасного порога — они показывают более широкое распространение вещества;
- разность итоговых полей для высокой и базовой диффузии.

Это напрямую иллюстрирует вывод: увеличение коэффициента диффузии приводит к более быстрому распространению вещества по помещению и выравниванию концентрационного поля.

## 11. Случайные возмущения

В качестве возмущающих воздействий в сценариях используются события с временем начала, длительностью и интенсивностью:

- открывание двери — временно увеличивает вентиляционный сток и формирует дополнительный поток около двери;
- изменение направления воздушных потоков — меняет знак или добавляет компоненту скорости;
- локальные сквозняки — усиливают скорость воздуха только в заданной области помещения;
- изменение температуры — меняет эффективную диффузию `D`;
- изменение давления внешней среды — изменяет интенсивность источника `Q` и вентиляционный режим.

## 12. Метрики переходных процессов

Для каждого сценария дополнительно рассчитываются:

- `meanC(t)` — средняя концентрация в помещении;
- `maxC(t)` — максимальная концентрация;
- `dangerArea(t)` — площадь области, где концентрация выше опасного порога;
- `firstDangerTime` — первое время формирования опасной зоны;
- `safeModeTime` — первое время, начиная с которого концентрация удерживается ниже безопасного порога не менее заданного времени.

Эти показатели позволяют проверить выводы переходного анализа: увеличение расхода ускоряет формирование опасной зоны, усиление вентиляции ускоряет очистку помещения, рост диффузии ускоряет распространение и выравнивание поля, а воздушные потоки делают поле концентрации асимметричным.
